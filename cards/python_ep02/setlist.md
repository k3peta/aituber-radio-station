# if文の基本
[type: talk]

まずは基本の形っす。

if age >= 18: {イフ、エイジ、大なりイコール、18、コロン。}

これで「もし年齢が18以上なら」っていう意味になるっす。
コロンを忘れちゃダメっすよ。Pythonはコロンが大事っす。 {コロンを忘れちゃダメっすよ。パイソンはコロンが大事っす。}

そして次の行をインデント、つまり4つスペースを入れるっす。

if age >= 18: {イフ、エイジ、大なりイコール、18、コロン。}
    print("成人です") {スペース4つ、プリント、かっこ、成人です、かっこ閉じ。}

Pythonはインデントでブロックを表すっす。 {パイソンはインデントでブロックを表すっす。}
波括弧じゃなくて、スペースで構造を作るんすよ。

# elseで「そうじゃなかったら」
[type: talk]

ifだけだと「もし〜なら」しかできないっすよね。 {イフだけだと「もし〜なら」しかできないっすよね。}
「そうじゃなかったら」を書きたい時はelseっす。 {「そうじゃなかったら」を書きたい時はエルスっす。}

if age >= 18: {イフ、エイジ、大なりイコール、18、コロン。}
    print("成人です") {スペース4つ、プリント、成人です。}
else: {エルス、コロン。}
    print("未成年です") {スペース4つ、プリント、未成年です。}

18歳以上なら「成人です」、それ以外なら「未成年です」っす。
怪談ちゃんは16歳だから……未成年っすね。ふふ。

# elifで条件を増やす
[type: talk]

条件が2つ以上ある時はelifっす。else ifの略っすよ。 {条件が2つ以上ある時はエリフっす。エルス、イフの略っすよ。}

if score >= 90: {イフ、スコア、大なりイコール、90、コロン。}
    print("S") {プリント、エス。}
elif score >= 70: {エリフ、スコア、大なりイコール、70、コロン。}
    print("A") {プリント、エー。}
elif score >= 50: {エリフ、スコア、大なりイコール、50、コロン。}
    print("B") {プリント、ビー。}
else: {エルス、コロン。}
    print("C") {プリント、シー。}

上から順番に条件をチェックして、最初に当てはまったところが実行されるっす。
90点以上ならS、70点以上ならA、50点以上ならB、それ以外はCっす。

# 比較演算子
[type: talk]

ifの条件で使う記号をまとめるっす。 {イフの条件で使う記号をまとめるっす。}

==は「等しい」っす。イコール2つっすよ。 {イコールイコールは「等しい」っす。イコール2つっすよ。}
1つだと代入になっちゃうから注意っす。
!=は「等しくない」っす。 {ビックリマーク、イコールは「等しくない」っす。}
>は「より大きい」、<は「より小さい」。 {大なりは「より大きい」、小なりは「より小さい」。}
>=は「以上」、<=は「以下」っす。 {大なりイコールは「以上」、小なりイコールは「以下」っす。}

これさえ覚えれば、いろんな条件が書けるっすよ。

# andとorで条件を組み合わせ
[type: talk]

条件を組み合わせたい時もあるっすよね。

andは「両方とも満たす時」っす。 {アンドは「両方とも満たす時」っす。}

if age >= 18 and age < 65: {イフ、エイジ、大なりイコール、18、アンド、エイジ、小なり、65、コロン。}
    print("現役世代") {プリント、現役世代。}

orは「どちらか一方でも満たす時」っす。 {オアは「どちらか一方でも満たす時」っす。}

if day == "土" or day == "日": {イフ、デイ、イコールイコール、土、オア、デイ、イコールイコール、日、コロン。}
    print("休日") {プリント、休日。}

英語そのままっすよね。andは「かつ」、orは「または」っす。 {英語そのままっすよね。アンドは「かつ」、オアは「または」っす。}
