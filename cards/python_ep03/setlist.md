# forループの基本
[type: talk]

まずはforループっす。 {まずはフォーループっす。}
「この中のものを順番に1つずつ処理する」っていう仕組みっす。

for i in range(5): {フォー、アイ、イン、レンジ、5、コロン。}
    print(i) {スペース4つ、プリント、アイ。}

これで0、1、2、3、4が順番に表示されるっす。
rangeの5は「5回繰り返す」って意味っすよ。 {レンジの5は「5回繰り返す」って意味っすよ。}
0から始まるのがポイントっす。

# rangeの使い方
[type: talk]

rangeはいろんな書き方ができるっす。 {レンジはいろんな書き方ができるっす。}

range(5)は0から4まで。 {レンジ5は0から4まで。}
range(1, 6)は1から5まで。 {レンジ1、6は1から5まで。}
range(0, 10, 2)は0、2、4、6、8。2ずつ飛ばすっす。 {レンジ0、10、2は0、2、4、6、8。2ずつ飛ばすっす。}

3番目の数字はステップっす。何個ずつ進むかっすよ。

# 文字列やリストをforで回す
[type: talk]

forは数字だけじゃなく、文字列もリストも回せるっす。 {フォーは数字だけじゃなく、文字列もリストも回せるっす。}

for char in "hello": {フォー、キャラ、イン、ハロー、コロン。}
    print(char) {スペース4つ、プリント、キャラ。}

これでh、e、l、l、oが1文字ずつ表示されるっす。
文字列は1文字ずつの集まりっすからね。

# whileループ
[type: talk]

もう1つのループがwhileっす。 {もう1つのループがワイルっす。}
「条件が満たされている間、ずっと繰り返す」っていう仕組みっす。

count = 0 {カウント、イコール、0。}
while count < 5: {ワイル、カウント、小なり、5、コロン。}
    print(count) {スペース4つ、プリント、カウント。}
    count += 1 {スペース4つ、カウント、プラスイコール、1。}

countが5より小さい間、ずっと繰り返すっす。 {カウントが5より小さい間、ずっと繰り返すっす。}
count += 1を忘れると永遠に終わらないから注意っすよ！ {カウント、プラスイコール、1を忘れると永遠に終わらないから注意っすよ！}

# breakとcontinue
[type: talk]

ループの途中で抜けたい時はbreakっす。 {ループの途中で抜けたい時はブレイクっす。}

for i in range(10): {フォー、アイ、イン、レンジ、10、コロン。}
    if i == 5: {イフ、アイ、イコールイコール、5、コロン。}
        break {ブレイク。}
    print(i) {プリント、アイ。}

これで0から4まで表示されて、5の時にループを抜けるっす。

continueは「今の回をスキップして次へ」っす。 {コンティニューは「今の回をスキップして次へ」っす。}

for i in range(5): {フォー、アイ、イン、レンジ、5、コロン。}
    if i == 2: {イフ、アイ、イコールイコール、2、コロン。}
        continue {コンティニュー。}
    print(i) {プリント、アイ。}

2だけスキップされて、0、1、3、4が表示されるっす。

# 実践：九九の表
[type: talk]

じゃあ実践っす。forを2重にして九九の表を作るっすよ。

for i in range(1, 10): {フォー、アイ、イン、レンジ、1、10、コロン。}
    for j in range(1, 10): {フォー、ジェイ、イン、レンジ、1、10、コロン。}
        print(i * j, end=" ") {プリント、アイ、かける、ジェイ、エンドイコール、スペース。}
    print() {プリント。}

forの中にforを入れるのをネストって言うっす。
2重ループは最初ちょっとむずかしいっすけど、慣れたら便利っす。
