# BGMをつける
[type: talk]

番組にBGMをつけると一気にラジオっぽくなるっす。 {番組にビージーエムをつけると一気にラジオっぽくなるっす。}

セグメントの中に角括弧でbgmタグを書くっす。 {セグメントの中に角括弧でビージーエムタグを書くっす。}

パスはmedia/bgm.mp3のように相対パスで書くっす。 {パスはメディア、スラッシュ、ビージーエムドットエムピースリーのように相対パスで書くっす。}
mediaフォルダの中のファイルを指定するっすよ。 {メディアフォルダの中のファイルを指定するっすよ。}
相対パスにしておけば、ローカルでもステーションでも動くっす。

音量の調整もできるっす。
[volume: 0.2]で音量20%っす。0.1から0.5くらいが聴きやすいっすよ。 {ボリューム、コロン、0.2で音量20パーセントっす。0.1から0.5くらいが聴きやすいっすよ。}

BGMを止めたい時は[bgm: none]っす。 {ビージーエムを止めたい時は、ビージーエム、コロン、ナンっす。}

# ジングル（区切り音）
[type: talk]

セグメントの合間にジングルを入れると、番組っぽさが増すっす。

mediaフォルダにjingle1.mp3、jingle2.mp3を入れておくと、 {メディアフォルダにジングル1ドットエムピースリー、ジングル2ドットエムピースリーを入れておくと、}
セグメントの切り替え時に自動で再生されるっす。

ジングル画像もjingle1.jpg {ジングル1ドットジェイペグ}を入れておけば、ジングル再生中に表示されるっすよ。

# フロート画像の使い方
[type: talk]

図やスライドを表示したい時はfloatタグっす。 {図やスライドを表示したい時はフロートタグっす。}

[float: media/slide.png]で画像が表示されるっす。 {フロート、コロン、メディア、スラッシュ、スライドドットピングで画像が表示されるっす。}
キャラクターの横にふわっと出てくるっすよ。

消すときは[float: none]っす。 {消すときはフロート、コロン、ナンっす。}

学習系の番組で「ここは図を見せたい」って時にすごく便利っす。
この講座でも使ってるっすよ。

# 背景画像の変更
[type: talk]

背景を変えたい時はbgタグっす。 {背景を変えたい時はバックグラウンドタグっす。}

[bg: media/background.jpg]で背景が変わるっす。 {ビージー、コロン、メディア、スラッシュ、バックグラウンドドットジェイペグで背景が変わるっす。}

オープニングとエンディングで違う背景にしたりできるっすよ。
mediaフォルダにopening.jpgとending.jpgを入れておけば、 {メディアフォルダにオープニングドットジェイペグとエンディングドットジェイペグを入れておけば、}
自動的にオープニング画像とエンディング画像として使われるっす。

# 感情タグ
[type: talk]

キャラクターの表情を変えたい時は感情タグを使うっす。

[emotion: happy]で笑顔、[emotion: sad]で悲しい顔、[emotion: angry]で怒った顔っすよ。 {エモーション、ハッピーで笑顔、エモーション、サッドで悲しい顔、エモーション、アングリーで怒った顔っすよ。}
[emotion: neutral]でデフォルトに戻るっす。 {エモーション、ニュートラルでデフォルトに戻るっす。}

VRMモデルにブレンドシェイプが設定されていれば、表情が変わるっす。 {ブイアールエムモデルにブレンドシェイプが設定されていれば、表情が変わるっす。}

# AIで読み指定を自動でつける
[type: talk]

英語や技術用語の多い台本は、読み指定をつけるのが大変っすよね。
そこでAIに「この台本の英語や技術用語に波括弧で読みを追加して」ってお願いするんっす。 {そこでエーアイに「この台本の英語や技術用語に波括弧で読みを追加して」ってお願いするんっす。}

たとえば、
「SELECT * FROM usersを解説します」っていう文を渡すと、
「SELECT * FROM usersを解説します。{セレクト、アスタリスク、フロム、ユーザーズを解説します。}」
って返してくれるっす。

これでTTSが変な読み方をしなくなるっす。 {これでティーティーエスが変な読み方をしなくなるっす。}

# AIで感情タグを自動付与
[type: talk]

もう一つのtips。AIに感情タグを自動でつけてもらう方法っす。 {もう一つのティップス。エーアイに感情タグを自動でつけてもらう方法っす。}

台本を渡して「セリフの感情に合わせてemotionタグを挿入して」って頼むんっす。 {台本を渡して「セリフの感情に合わせてエモーションタグを挿入して」って頼むんっす。}

楽しい話題の前に[emotion: happy]、 {楽しい話題の前にエモーション、ハッピー、}

を自動で入れてくれるっすよ。

手作業でやると大変だけど、AIなら一瞬っす。 {手作業でやると大変だけど、エーアイなら一瞬っす。}
