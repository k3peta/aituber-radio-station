# AITuber Radioって何？
[type: talk]

AITuber Radioは、VRMキャラクターがラジオ番組をやってくれるアプリっす。 {エーアイチューバーレディオは、ブイアールエムキャラクターがラジオ番組をやってくれるアプリっす。}
台本を書いたら、キャラクターが声付きで読み上げてくれるっす。

Chrome拡張機能として動くっすよ。 {クローム拡張機能として動くっすよ。}
ブラウザがあれば使えるっす。

できることは、
1、VRMキャラクターの表示。 {1、ブイアールエムキャラクターの表示。}
2、VOICEVOXを使った音声読み上げ。 {2、ボイスボックスを使った音声読み上げ。}
3、BGMやジングルの再生。 {3、ビージーエムやジングルの再生。}
4、図やスライドの表示。
5、録画機能。

一人でラジオ番組が作れちゃうっす！

# 必要なもの
[type: talk]

用意するものは2つっす。

1つ目、Google Chromeブラウザ。 {1つ目、グーグルクロームブラウザ。}
2つ目、VOICEVOX。音声合成ソフトっす。無料でダウンロードできるっすよ。 {2つ目、ボイスボックス。音声合成ソフトっす。無料でダウンロードできるっすよ。}

デフォルトで怪談ちゃんのキャラクターが入ってるので、VRMモデルは最初はなくても大丈夫っす。 {デフォルトで怪談ちゃんのキャラクターが入ってるので、ブイアールエムモデルは最初はなくても大丈夫っす。}
自分のキャラクターを使いたくなったら、VRoid Hubからダウンロードしたり自分で作ったりできるっすよ。 {自分のキャラクターを使いたくなったら、ブイロイドハブからダウンロードしたり自分で作ったりできるっすよ。}

# VOICEVOXのインストール
[type: talk]

VOICEVOXの公式サイトからダウンロードするっす。 {ボイスボックスの公式サイトからダウンロードするっす。}
Mac版とWindows版があるっすよ。 {マック版とウインドウズ版があるっすよ。}

インストールしたら起動するっす。
起動すると、自動的にエンジンが立ち上がるっす。
画面が出れば準備OKっす。 {画面が出れば準備オーケーっす。}

大事なのは、AITuber Radioを使う時はVOICEVOXを先に起動しておくことっす。 {大事なのは、エーアイチューバーレディオを使う時はボイスボックスを先に起動しておくことっす。}

# 拡張機能のインストール
[type: talk]

AITuber Radioの拡張機能をChromeに入れるっす。 {エーアイチューバーレディオの拡張機能をクロームに入れるっす。}

GitHubからリポジトリをダウンロードするっす。 {ギットハブからリポジトリをダウンロードするっす。}
ZIPでダウンロードしてもいいし、git cloneしてもいいっすよ。 {ジップでダウンロードしてもいいし、ギットクローンしてもいいっすよ。}

次に、Chromeの拡張機能ページを開くっす。 {次に、クロームの拡張機能ページを開くっす。}
アドレスバーにchrome://extensionsと入力するっす。 {アドレスバーにクローム、コロン、スラッシュスラッシュ、エクステンションズと入力するっす。}

右上の「デベロッパーモード」をオンにするっす。
「パッケージ化されていない拡張機能を読み込む」を押すっす。
ダウンロードしたフォルダを選択するっす。

これで拡張機能がインストールされるっす！

# 最初の起動
[type: talk]

拡張機能のアイコンをクリックすると、ポップアップが開くっす。
ここから「ビューアを開く」ボタンを押すっす。

新しいタブでビューアが開くっす。
デフォルトで怪談ちゃんのキャラクターと背景が自動的に読み込まれるっす。
これはサンプルなので、自分のキャラクターに変えたい場合はポップアップから読み込むっすよ。

# VRMモデルの読み込み
[type: talk]

自分のキャラクターを使いたい時は、ポップアップからVRMファイルを読み込むっす。 {自分のキャラクターを使いたい時は、ポップアップからブイアールエムファイルを読み込むっす。}
ポップアップの「VRMを読み込む」ボタンから選択するっすよ。 {ポップアップの「ブイアールエムを読み込む」ボタンから選択するっすよ。}

読み込むと、3Dキャラクターがデフォルトから変わるっす。
これが番組のパーソナリティになるっすよ。

# 最初の再生テスト
[type: talk]

大事なのは、VOICEVOXが起動してることっす。 {大事なのは、ボイスボックスが起動してることっす。}

ポップアップの下の方にテキスト入力欄があるっす。
そこにテキストを入力して送信ボタンを押すっす。

キャラクターがテキストを読み上げてくれたら成功っす！
口パクもするっすよ。かわいいっす。

# ステーションを使う時のひと手間
[type: talk]

AITuber Radio Stationのサイトから番組を再生する時は、ひと手間必要っす。 {エーアイチューバーレディオステーションのサイトから番組を再生する時は、ひと手間必要っす。}

ポップアップを開くと、拡張機能IDが表示されてるっす。 {ポップアップを開くと、拡張機能アイディーが表示されてるっす。}
これをコピーして、ステーションのページにあるテキストボックスに貼り付けるっす。
「接続」ボタンを押すと、ステーションと拡張機能がつながるっすよ。

これで好きなカードの「再生」ボタンを押すだけで番組が始まるっす！

# 今日のまとめ
[type: talk]

はいっ、第1回のまとめっす。

1、VOICEVOXをインストールして起動。 {1、ボイスボックスをインストールして起動。}
2、拡張機能をChromeに読み込む。 {2、拡張機能をクロームに読み込む。}
3、ビューアを開く。デフォルトキャラが自動で読み込まれる。
4、自分のキャラクターはポップアップから読み込む。
5、ステーション利用時は拡張機能IDをコピーして接続。 {5、ステーション利用時は拡張機能アイディーをコピーして接続。}

これでセットアップ完了っす！
