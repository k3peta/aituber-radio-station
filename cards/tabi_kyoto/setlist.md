# 路地裏1
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.7]
京都って、大通りから一本入るだけで全然違う世界が広がるんだよね。
特におすすめなのが、東山エリアの「ねねの道」から「石塀小路」に入るルート。
石塀小路は、高台寺の南側にある細い路地で、石畳の道の両側に料亭やお茶屋さんの板塀が続くの。
幅は2メートルくらいしかなくて、人ひとりがやっと通れるくらい。
夕方に行くと、料亭の灯りがほんのり漏れてきて、もう時代劇みたいな雰囲気。
観光客はみんな清水寺や八坂神社に行くから、ここは意外と静かなんだよね。

# グルメ1
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: happy]
[intensity: 0.7]
京都のグルメで外せないのが、やっぱりおばんざい。
おばんざいって、京都の家庭料理のことなんだけど、素材の味を生かしたシンプルな料理が多いの。
ひじきの煮物とか、万願寺とうがらしの焼き浸しとか、お揚げさんと壬生菜の炊いたんとか。
「炊いたん」っていうのは京都弁で「煮物」のこと。
錦市場の近くには、カウンターでおばんざいを出してくれるお店がたくさんあって。
一人旅でもふらっと入れるのが嬉しいんだよね。

# 旅のコツ1
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.7]
京都を旅するなら、移動手段は断然バスと電車の組み合わせがおすすめ。
バス一日券は700円で乗り放題だから、まずはこれを買おう。
ただし、紅葉シーズンとか桜の時期はバスがものすごく混むから、地下鉄を使うのが賢い。
特に四条通りと河原町通りは渋滞がひどいから、この区間は阪急電車か地下鉄で移動するといいよ。

# 旅のコツ2
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.6]
それから、京都のベストシーズンについて。
春の桜と秋の紅葉が王道だけど、実は真冬の京都もすごくいいの。
観光客が少ないから、金閣寺も清水寺も独り占めできることがある。
そして雪が降った日の京都は、もう別世界。
雪化粧した金閣寺は「これが見たかったんだ」って感動するレベル。
冬は旅館の料金も安いし、湯豆腐が最高に美味しい季節だしね。

最後に一つ。京都旅では「朝を制する者が京都を制する」っていうのがあたしの持論。

レトロな雰囲気の中で飲む京の朝コーヒー、最高なんだよなぁ。
