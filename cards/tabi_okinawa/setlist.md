# 海1
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: happy]
[intensity: 0.7]
沖縄の海って、本当に色が違うんだよね。
本島でも十分綺麗だけど、おすすめは読谷村の「ニライビーチ」。
大きなリゾートホテルの敷地内にあるんだけど、ビーチ自体は誰でも入れるの。
白い砂浜が弧を描いていて、水の透明度がすごい。
干潮のときは沖のほうまで歩いて行けて、潮だまりで熱帯魚が見れたりする。
シュノーケルなしでも魚が見えるくらい透明なんだよ。

# グルメ1
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: happy]
[intensity: 0.7]
沖縄グルメといえば、まずはソーキそば。
ソーキっていうのは豚のスペアリブのことで、とろとろに煮込んだソーキが太めのそばの上にどーんと乗ってるの。
出汁はカツオと豚骨のブレンドで、見た目よりあっさり。
お店によって全然味が違うから、食べ比べるのが楽しいんだよね。
あたしのおすすめは首里の「首里そば」。手打ち麺のコシがすごくて、スープは透明感のある上品な味。
11時開店で、麺がなくなったら終了だから、早めに行くのがコツ。

それからタコライス。
タコスの具材をごはんの上に乗せた料理で、沖縄生まれのソウルフードなの。
ひき肉、レタス、トマト、チーズがごはんの上にこんもり。
発祥は金武町の「キングタコス」っていうお店で、米軍基地の近くにあるの。
ボリュームがすごくて、一皿で大満足。しかも安い。
地元の人は「キンタコ」って呼んでて、注文は「タコライスチーズ野菜」が定番。
辛いサルサソースをかけると、もう止まらなくなるよ。

口を開けてるのが「阿」、閉じてるのが「吽」。

壺屋通りにはシーサーの工房がたくさんあって、自分だけのシーサーを作る体験もできるよ。
手作りのシーサーって、味があっていいんだよね。
