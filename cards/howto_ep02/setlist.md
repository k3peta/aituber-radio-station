# Deck（カード）から番組を選ぶ
[type: talk]
[status: 📦 Deckボタンを押すだけっす！]

拡張機能のポップアップを開くと、カラフルな「番組カード」が並んでるっすよね？
これがDeckっす。

好きなカードをポチッと押すだけで、番組の台本・BGM・画像が自動的にダウンロードされて、すぐに放送が始まるっす！
難しい設定は一切不要っすよ。

GitHubにある「Radio Station」から最新の番組が配信されてるから、放っておいても番組が増えていくっす。

# フリートークボタン
[type: talk]
[status: 🗣️ フリートークを試してみてっす！]

「台本が決まってないけど、とりあえず喋らせたい！」って時は「フリートーク」ボタンっす！

これを押すと、AIがランダムな話題を選んで、その場で面白い話を考えてくれるっす。
話題の数は今や300近くあるっすよ。

最新バージョンでは、日付に合わせて「節分」や「バレンタイン」の話もしてくれるようになったっす！
季節感のあるラジオ、いいっすよね。

# エンドレストーク（連鎖フリートーク）
[type: talk]
[status: 🔄 ずっと喋り続けるモードっす！]

「作業用BGMとしてずっと流しておきたい」という時に最高なのが「エンドレストーク」ボタンっす！

一度押すと、AIが話題を次々と選んで、あなたが止めるまで延々と喋り続けてくれるっす。
話題の間にはBGMも自動で流れるから、本当の24時間放送ラジオみたいになるっすよ。

止める時は「停止」ボタンを押せばいつでも終われるっす。

# 自動コメント応答
[type: talk]
[status: 💬 コメントに返事をするっすよ！]

セットアップ画面で「わんコメ」と連携するか、ビューアの下にある「ローカルチャット」をオンにしてみてっす。

リスナーさんがコメントをくれると、AIが名前を呼んで返事をしてくれるっす！
「怪談ちゃん、こんにちは！」って送られたら、「こんにちはっす！今日も見に来てくれて嬉しいっす！」って返してくれるっすよ。

2人以上のキャラがいる時は、コメントで名前を呼ばれた方のキャラが返事をする賢い機能も付いてるっす。

# 今日のまとめ
[type: talk]
[status: ✅ 第2回のまとめっす！]

はいっ、第2回のまとめっす。

1、ポップアップのカード（Deck）を押すだけで番組が始まる。
2、フリートークボタンでAIが話題を選んで喋る。
3、エンドレストークなら、止めるまでずっと24時間放送。
4、コメントを入れると、名前を呼んでAIが返事をしてくれる。

これだけで、もう立派なVTuber配信っすね！
