# セットリストの基本構造
[type: talk]

台本はマークダウン形式で書くっす。拡張子は.mdっす。 {台本はマークダウン形式で書くっす。かくちょうしはドットエムディーっす。}

最初にYAMLフロントマターっていうヘッダーを書くっす。 {最初にヤムルフロントマターっていうヘッダーを書くっす。}

3つのハイフンで始めて、設定を書いて、また3つのハイフンで閉じるっす。

タイトル、話者ID、話速、セリフ間の間隔を設定できるっす。

speakerは話者のIDっす。VOICEVOXの話者番号を指定するっすよ。 {スピーカーは話者のアイディーっす。ボイスボックスの話者番号を指定するっすよ。}
speedは話す速さっす。1.0が標準で、0.95だとちょっとゆっくりっす。

# セグメントの書き方
[type: talk]

番組はセグメントに分かれるっす。コーナーみたいなもんっすよ。

シャープをつけてセグメント名を書くっす。
その下にタイプを指定するっす。

# オープニング、改行、[type: talk]、みたいな感じっす。 {シャープ、オープニング、改行、タイプ、トーク、みたいな感じっす。}

そして、その下にセリフを書いていくっす。
1行が1つのセリフになるっす。 {いちぎょうが1つのセリフになるっす。}

空行を入れると、そこで少し間が空くっす。

# 読み指定の書き方
[type: talk]

英語や記号が入ったテキストは、TTS（音声合成）が変な読み方をすることがあるっす。 {英語や記号が入ったテキストは、ティーティーエス（音声合成）が変な読み方をすることがあるっす。}

そんな時は「読み指定」を使うっす。

波括弧の中に読み方を書くっす。
「SELECT * FROM users」の後に、波括弧で「セレクト、アスタリスク、フロム、ユーザーズ」って書くっすよ。 {セレクト、アスタリスク、フロム、ユーザーズの後に、波括弧で「セレクト、アスタリスク、フロム、ユーザーズ」って書くっすよ。}

すると、画面には英語が表示されて、読み上げはカタカナで行われるっす。
技術系の番組では必須の機能っすよ。

# VOICEVOXの話者を選ぶ
[type: talk]

VOICEVOXにはたくさんの声があるっす。 {ボイスボックスにはたくさんの声があるっす。}

ずんだもんが3番、四国めたんが2番、九州そらが16番っす。
怪談ちゃんは「ずんだもん（ひそひそ声）」の38番を使ってるっすよ。

番号はVOICEVOXの画面で確認できるっす。 {番号はボイスボックスの画面で確認できるっす。}
好きな声を選んでほしいっす。

# 実際に台本を書いてみる
[type: talk]

最小限の台本はこんな感じっす。

3つのハイフンでヘッダーを開始、
タイトルをつけて、
スピーカーを38に設定、
3つのハイフンで閉じて、
シャープでセグメント名を書いて、
タイプをトークに設定して、
セリフを書くっす。

これだけで番組が作れるっす！

# フォルダから読み込む方法
[type: talk]

台本ファイルができたら、読み込むっす。

ビューアの画面に「フォルダを読み込む」ボタンがあるっす。
台本とメディアファイルが入ったフォルダを選択するっすよ。

フォルダの中に、
セットリストのファイル、
readingsフォルダに朗読用テキストファイル、 {リーディングズフォルダに朗読用テキストファイル、}
mediaフォルダにBGMやジングル、 {メディアフォルダにビージーエムやジングル、}
を入れておくっす。

「再生」ボタンを押すと番組が始まるっす！

# ステーションから再生する方法
[type: talk]

もっと簡単な方法もあるっす。ステーションっす。

AITuber Radio Stationのサイトにアクセスすると、公開されてる番組がカード形式で並んでるっす。 {エーアイチューバーレディオステーションのサイトにアクセスすると、公開されてる番組がカード形式で並んでるっす。}
好きなカードの「再生」ボタンを押すだけっす。

台本もメディアも全部ステーションから自動でダウンロードされるっすよ。

# 今日のまとめ
[type: talk]

はいっ、第2回のまとめっす。

1、台本はマークダウン形式で書く。
2、ヘッダーで話者や速度を設定。
3、シャープでセグメントを区切る。
4、波括弧で読み指定ができる。
5、フォルダ読み込みかステーションから再生。
