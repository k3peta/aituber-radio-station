# リストの基本
[type: talk]

リストは角括弧で作るっす。

fruits = ["りんご", "みかん", "ぶどう"] {フルーツ、イコール、かぎかっこ、りんご、みかん、ぶどう、かぎかっこ閉じ。}

3つの果物が1つのリストに入ったっす。
数字でもOKっす。

numbers = [10, 20, 30, 40, 50] {ナンバーズ、イコール、かぎかっこ、10、20、30、40、50、かぎかっこ閉じ。}

文字列と数字を混ぜることもできるっすけど、普通は同じ種類を入れるっす。

# インデックスでアクセス
[type: talk]

リストの中身を取り出すには、インデックスを使うっす。番号っすよ。

fruits[0] {フルーツ、かぎかっこ、0、かぎかっこ閉じ。}

これで「りんご」が取れるっす。
大事なのは、番号は0から始まるってことっす。
fruits[0]がりんご、fruits[1]がみかん、fruits[2]がぶどうっす。 {フルーツ0がりんご、フルーツ1がみかん、フルーツ2がぶどうっす。}

マイナスも使えるっすよ。fruits[-1]で最後の要素、ぶどうが取れるっす。 {マイナスも使えるっすよ。フルーツ、マイナス1で最後の要素、ぶどうが取れるっす。}

# リストの追加と削除
[type: talk]

リストに要素を追加するにはappendっす。 {リストに要素を追加するにはアペンドっす。}

fruits.append("いちご") {フルーツドットアペンド、いちご。}

これでリストの最後に「いちご」が追加されるっす。

削除はremoveっす。 {削除はリムーブっす。}

fruits.remove("みかん") {フルーツドットリムーブ、みかん。}

みかんが消えるっす。存在しない値を指定するとエラーになるから注意っすよ。

# リストの長さと存在チェック
[type: talk]

リストの長さはlen関数で取れるっす。 {リストの長さはレン関数で取れるっす。}

len(fruits) {レン、フルーツ。}

果物が3つ入ってたら3が返るっす。

リストに特定の値があるか確認するにはinを使うっす。 {リストに特定の値があるか確認するにはインを使うっす。}

if "りんご" in fruits: {イフ、りんご、イン、フルーツ、コロン。}
    print("りんごあるっす！") {プリント、りんごあるっす。}

シンプルで分かりやすいっすよね。

# リストとforループ
[type: talk]

リストとforの組み合わせは最強っす。 {リストとフォーの組み合わせは最強っす。}

for fruit in fruits: {フォー、フルート、イン、フルーツ、コロン。}
    print(fruit) {スペース4つ、プリント、フルート。}

リストの中身を1つずつ取り出して表示するっす。
forに直接リストを渡せるのがPythonの便利なところっすよ。 {フォーに直接リストを渡せるのがパイソンの便利なところっすよ。}

番号付きで取り出したい時はenumerateっす。 {番号付きで取り出したい時はエニュメレイトっす。}

for i, fruit in enumerate(fruits): {フォー、アイ、カンマ、フルート、イン、エニュメレイト、フルーツ、コロン。}
    print(i, fruit) {プリント、アイ、フルート。}

0 りんご、1 みかん、みたいに番号付きで出るっす。

# リストのスライス
[type: talk]

リストの一部だけ取り出すこともできるっす。スライスって言うっすよ。

numbers = [10, 20, 30, 40, 50] {ナンバーズ、イコール、10、20、30、40、50。}
numbers[1:3] {ナンバーズ、かぎかっこ、1コロン3、かぎかっこ閉じ。}

これで20と30が取れるっす。
1番目から3番目の手前までっす。手前っていうのがポイントっすよ。

numbers[:3]は最初から3つ。 {ナンバーズ、コロン3は最初から3つ。}
numbers[2:]は2番目から最後まで。 {ナンバーズ、2コロンは2番目から最後まで。}
