# 海の幸1 — 寒ブリ
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: happy]
[intensity: 0.8]
富山グルメで最初に語りたいのは、やっぱり氷見の寒ブリ！
氷見って富山県の北西にある港町なんだけど、冬になると脂の乗った天然ブリが大量に揚がるの。
特に12月から2月にかけてが最盛期で、「ひみ寒ぶり宣言」っていう公式の宣言が出るんだよ。
刺身にすると脂がとろっとろで、醤油をはじくくらい。
ブリしゃぶにすると、お湯にくぐらせた瞬間にぶわっと脂の膜が浮いてきて、口に入れるともうホロッホロ。
氷見の「きときと寿司」で食べるブリの握りは、もう言葉が出ないくらい美味しいんだよね。

# 海の幸2 — 白えび
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.7]
続いて富山といえば、白えび！
「富山湾の宝石」って呼ばれてるの、知ってた？
透き通ったピンク色の小さなえびで、富山湾でしかまとまった量が獲れないんだよ。
白えびの刺身は、ねっとりとした甘さが口の中に広がって、もう本当に上品。
でもあたしのイチオシは白えびのかき揚げ！
サクサクの衣の中にぷりっぷりの白えびがぎゅっと詰まっていて。
そのまま食べても美味しいし、天つゆにつけても、塩でも最高。
富山駅の「白えび亭」で食べる白えび天丼は、富山に着いたら真っ先に行ってほしいお店だよ。

# ソウルフード1 — ますのすし
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.7]
富山のソウルフードといえば、まず「ますのすし」！
笹の葉で包まれた円形の押し寿司で、中にはピンク色のマスと酢飯がぎっしり。
富山駅のお土産ランキングでは不動の一位だよ。
でもね、実はますのすしって、お店によって全然味が違うの。
「源」は定番中の定番で安定の美味しさ。
「青山総本舗」はマスの厚みが段違いで、食べごたえ抜群。
「扇一」は昔ながらの製法で、酢の加減が絶妙。
富山通の間では「推しのますのすし屋」があるくらい、奥が深い世界なんだよ。

# ソウルフード2 — 富山ブラック
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: surprised]
[intensity: 0.8]
次に語りたいのが「富山ブラック」！
名前の通り、スープが真っ黒なラーメンなの。
元祖は「大喜」っていうお店で、昭和20年代に汗をかく労働者のためのおかず的なラーメンとして誕生したらしい。
だからとにかくしょっぱい！最初は「えっ」ってなるくらい塩気が強いの。
でもこれが癖になるんだよね〜。
真っ黒いスープに太めの麺が絡んで、上にはメンマとチャーシューと粗びきの黒コショウ。
白ごはんと一緒に食べるのが富山流。ラーメンで白飯を食べるっていう、ある意味最強の炭水化物コンボだよ。

讃岐うどんともきしめんとも違う、独特のつるっとした食感。
冷たいざるうどんで食べると、小麦の甘みがダイレクトに伝わってきて、本当に美味しい。
地元では温かいうどんに細ネギと生姜を添えて食べるのが定番。
寒い冬の日に食べる氷見うどんは、体の芯から温まるよ。

# 旅のコツ1
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.7]
富山グルメ旅のベストシーズンは、ずばり冬！
12月から2月がおすすめだよ。
寒ブリは冬が旬だし、カニも美味しい季節。
富山湾越しに見える立山連峰の雪景色も最高に美しいの。
北陸新幹線で東京から2時間ちょっとで行けるから、週末旅にもぴったり。
金曜の夜に出発して、土日で食べまくる旅、あたしはこれを「グルメ修行」って呼んでるんだけどね。

# 旅のコツ2
[type: talk]
[bgm: media/bgm_chill.mp3]
[emotion: neutral]
[intensity: 0.6]
移動のコツなんだけど、富山市内は路面電車がすごく便利。

最後に一つ。富山旅では「回転寿司」を侮っちゃダメだよ。
富山の回転寿司は、他の地域の回転寿司とはレベルが段違い。
「すし玉」とか「きときと寿し」とか、地元の新鮮なネタを使ってるから、回転寿司なのに下手な高級寿司店より美味しかったりする。
しかもお値段はリーズナブル。一人3000円もあればお腹いっぱい食べられるよ。
あたしのおすすめは、白えび、バイ貝、のど黒のネタ。この3つは絶対頼んでね！
