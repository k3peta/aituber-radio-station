# 変数ってなに
[type: talk]

次は変数っす。図の右上にあるやつっす。
変数って、値を入れる箱みたいなもんっす。

ネーム、イコール、かぎかっこ、先輩、かぎかっこ閉じ。
って書くと、ネームっていう箱に「先輩」が入るっす。

そしたら、
プリント、かっこ、ネーム、かっこ閉じ。
って書くと、先輩って表示されるっす。

数字も入れられるっす。
エイジ、イコール、25。
とか。
……先輩の年齢は聞かないっすけどね。ふふ。

# 計算もできる
[type: talk]

パイソンは計算もできるっす。電卓みたいに使えるっす。
図の左下にあるやつっすね。

1たす1は2。
10かける3は30。
100わる7は……14.285714なんとかっす。

割り算の余りも出せるっす。パーセント記号を使うっす。
10パーセント3で、1が出るっす。

これ、プログラミングでめっちゃよく使うっすよ。
偶数か奇数か調べたりできるっす。
