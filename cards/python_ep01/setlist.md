# 変数ってなに
[type: talk]

次は変数っす。変数って、値を入れる箱みたいなもんっす。

name イコール かぎかっこ 先輩 かぎかっこ閉じ。
って書くと、nameっていう箱に「先輩」が入るっす。

そしたら、
print かっこ name かっこ閉じ
って書くと、先輩って表示されるっす。

数字も入れられるっす。
age イコール 25
とか。
……先輩の年齢は聞かないっすけどね。ふふ。

# 計算もできる
[type: talk]

Pythonは計算もできるっす。電卓みたいに使えるっす。

1 たす 1 は 2。
10 かける 3 は 30。
100 わる 7 は……14.285714なんとかっす。

割り算の余りも出せるっす。パーセント記号を使うっす。
10 パーセント 3 で、1が出るっす。

これ、プログラミングでめっちゃよく使うっすよ。
偶数か奇数か調べたりできるっす。
