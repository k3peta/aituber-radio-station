AITuber Radio Station

AITuber Radio の台本カードをワンクリックで再生。
Chrome 拡張機能と連携して、お好みの番組をローカルでお楽しみいただけます。

🔌 拡張機能の接続 未接続
AITuber Radio ポップアップの ID をコピーして貼り付けてください。

🛠️ はじめかた

1
拡張機能をインストール
AITuber Radio をChromeに追加
2
VOICEVOX を起動
音声合成エンジンをローカルで起動
3
IDをコピーして接続
ポップアップの📋コピーで ID を取得 → 上のテキストボックスに貼り付けて「接続」
4
▶ 再生を押す
好きなカードを選んでワンクリック再生！
カードを読み込み中...
